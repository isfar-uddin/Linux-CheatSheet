#Description

#### Create Directory

`mkdir directory_name`

#### Create File
`touch file_name`

#### Create file under a new directory

`mkdir directory_name && touch file_name`

#### Move file

`mv file_name directory_name`

#### Rename file name

`mv file_name new_file_name`
